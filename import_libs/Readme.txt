Import libraries for libzplay.dll:

libzplay.a		- import library for GNU C++/MinGW/Dev-Cpp
libzplay.lib		- import library for Microsoft VC++
libzplay_borland.lib	- import library for Borland BCC55 command-line compiler


For VC.NET, VB.NET, C# and Delphi there is no need for import library,
just include files:

- libzplayNET.h
- libzplay.vb
- libzplay.cs
- libzplay.pas
