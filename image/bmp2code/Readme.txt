This is small application for converting Windows bitmap into C++ code.
Application generates color table and bit table. User needs to include these tables
into C++ source file and compile. Application also generates source code for reading
function. This function will create HBITMAP from color table and bit table.
Read example.cpp file for information how to use this tables to create bitmap.
