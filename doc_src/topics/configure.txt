User can configure each <LINK libZPlayCpp::ZPlay, ZPlay> instance independently.
It is possible to create 2 instance and configure each instance differently.

<INCLUDE extensions\\note_start.txt>There is default value for each setting.
<INCLUDE extensions\\note_end.txt>
