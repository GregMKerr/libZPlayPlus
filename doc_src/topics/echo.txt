Summary:
Echo processor.

Description:
libZPlay implements programmable echo processor.
User can create multiple echo effects for each channel separately to produce advanced echo and reverb effect.

<INCLUDE extensions\\note_start.txt>By default, echo processor is disabled and simple 1000 ms delay is programmed.
<INCLUDE extensions\\note_end.txt>
