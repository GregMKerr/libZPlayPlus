Summary:
Choose wave-in device as audio input for ZPlay class.

Description:
User can specify wave-in device for audio input.
If you have multiple soundcards, you can choose input soundcard.

Using this feature, you can record samples from one soundcard, change these samples with
library equalizer and output samples on second soundcard.
