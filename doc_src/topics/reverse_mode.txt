Summary:
Reverse playing. Play song backward.

Description:
libZPlay library has one very special feature. This is <b>reverse playing</b>.

Library can perforn <b>real reverse</b> playing.
There is no predecoding of whole stream into memory, no large cache, no additional memory allocation, no temporary disk files ...
This is true reverse playing.
