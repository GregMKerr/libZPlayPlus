Summary:
Player volume is internal volume control.

Description:
Player volume control will change audio samples to control sound volume.
This control is specific for each instance of ZPlay and user can control each instance independently.

<IMAGE note> Note: This control will not affect master computer volume.

You can play 2 songs using 2 instances and control volume of each song by changing <i>player volume</i>.
