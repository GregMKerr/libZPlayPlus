Summary:
  Enumerate waveform-audio output devices.

Returns:
Number of waveform-audio output devices present in the system.
A return value of zero means that no devices are present or that an error occurred.

Remarks:
Use this function to determine number of waveform-audio output devices.
If you have multiple soundcards you can specify soundcard for audio output.
