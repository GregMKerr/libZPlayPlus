Summary:
Stream status informations.

Parameters:
fPlay         - If song is playing this value is nonzero. See <LINK libZPlayDelphi.ZPlay.StartPlayback, Play>.
fPause        - If song is paused this value is nonzero. See <LINK libZPlayDelphi.ZPlay.PausePlayback, PausePlayback>.
fEcho         - If echo is enabled this value is nonzero. See <LINK libZPlayDelphi.ZPlay.EnableEcho, EnableEcho>.
fEqualizer    - If equalizer is enabled this value is nonzero. See <LINK libZPlayDelphi.ZPlay.EnableEqualizer, EnableEqualizer>.	
fVocalCut     - If vocal cut is enabled this value is nonzero. See <LINK libZPlayDelphi.ZPlay.StereoCut, StereoCut>.
fSideCut      - If side cut is enabled this value is nonzero. See <LINK libZPlayDelphi.ZPlay.StereoCut, StereoCut>.
fChannelMix   - If channel mixing is enabled this value is nonzero. See <LINK libZPlayDelphi.ZPlay.MixChannels, MixChannels>.
fSlideVolume  - If song is in "fade" interval this value is nonzero. See <LINK libZPlayDelphi.ZPlay.SlideVolume, SlideVolume>.
nLoop         - Number of loop cycles to play. See <LINK libZPlayDelphi.ZPlay.PlayLoop, PlayLoop>.
fReverse      - If song is in reverse mode this value is nonzero. See <LINK libZPlayDelphi.ZPlay.ReverseMode, ReverseMode>.
nSongIndex    - Index of playing song, increasing for gapless plaing. See <LINK libZPlayDelphi.ZPlay.AddFile, AddFile>.
nSongsInQueue - Number of songs in gapless queue. See  See <LINK libZPlayDelphi.ZPlay.AddFile, AddFile>.
