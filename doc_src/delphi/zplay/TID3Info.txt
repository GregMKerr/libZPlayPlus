Summary:
ID3 informations.

Parameters:
Title    - Title.
Artist   - Artist.
Album    - Album.
Year     - Year.
Comment  - Comment
TrackNum - Track number.
Genre    - Genre.

Remarks:
All members of this structure are strings.
Class instance allocates memory for these members and class instance is responsible for this memory.
These pointers are valid until you destroy class.
